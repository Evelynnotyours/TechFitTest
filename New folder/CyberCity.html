Available Majors
Information and Communication Technology▶️

Information and Communication Technology (ICT) ဆိုတာ အချက်အလက်နဲ့ ဆက်သွယ်ရေးနည်းပညာတွေကို အခြေခံထားတဲ့ ပညာရပ်ဖြစ်ပြီး၊ ကွန်ပျူတာ၊ ဆက်သွယ်ရေးစနစ်၊ အင်တာနက်နဲ့ အခြားသော နည်းပညာတွေကို အသုံးပြုပြီး အချက်အလက် ကို စီမံခန့်ခွဲ၊ ထုတ်လုပ်၊ သိမ်းဆည်း၊ ဖလှယ်နိုင်စေတဲ့ နည်းပညာရပ်ပဲဖြစ်ပါတယ်။ ICT ထဲမှာ Software Development၊ Networking၊ Cybersecurity၊ Database Management စတဲ့ နယ်ပယ်တွေပါဝင်ပြီး အခေတ်မီ နည်းပညာနယ်ပယ်မှာ အရေးကြီးဆုံး အခြေခံအတန်းလည်း ဖြစ်ပါတယ်။

Career Path

1. Software Developer / Programmer
2. Network Administrator / Engineer
3. Database Administrator
4. Cybersecurity Specialist
5. System Analyst / IT Consultant
6. Cloud Computing Engineer

Electronic Engineering▶️

Electronic Engineering ဆိုတာ အီလက်ထရွန်းနစ် Circuitတွေ၊ အီလက်ထရွန်းနစ်ပစ္စည်းတွေ၊ အီလက်ထရွန်းနစ်system‌တွေကို တီထွင် ဒီဇိုင်းလုပ်တဲ့ အင်ဂျင်နီယာ ပညာရပ်ပဲဖြစ်ပါတယ်။ DC ကို အသုံးပြုတဲ့ အီလက်ထရွန်းနစ်Circuitတွေက စပြီး ဆက်သွယ်ရေးနည်းပညာ၊ ထိန်းချုပ်စနစ်တွေ၊ ကွန်ပျူတာချစ်ပ် ဒီဇိုင်းနဲ့ ထုတ်လုပ်ပုံတွေကိုလည်း လေ့လာရပါတယ်။

Career Path

1. Desktop Support Engineer
2. Service Engineer
3. Network Planning Engineer
4. Control Centre / Communication / System Analyst / Design Engineer

Advanced Materials Engineering▶️

Advanced Materials Engineering ဆိုတာ သေးငယ်သိပ်သိပ်ကျဲကျဲအဆင့်မှစပြီး သဘာဝနဲ့ အစွမ်းထက်အသစ်ထုတ်လုပ်ထားတဲ့ ပစ္စည်းတွေကို လေ့လာ၊ တီထွင်၊ ဒီဇိုင်းလုပ်တဲ့ ပညာရပ်ဖြစ်ပါတယ်။ Nanomaterials, Biomaterials, Composite Materials စတဲ့ နည်းပညာတွေပါဝင်ပြီး လေယာဉ်အင်ဂျင်နီယာ, ဆေးဘက်ဆိုင်ရာ, စက်မှုထုတ်လုပ်မှုနယ်ပယ်တွေအတွက် အရေးကြီးပါတယ်။

Career Path

1. Materials Scientist / Engineer
2. Nanotechnology Researcher
3. Polymer Engineer
4. Aerospace Materials Specialist
5. Biomedical Materials Developer
6. Manufacturing Process Engineer

Precision Engineering▶️

Precision Engineering ဆိုတာ အလွန်တိကျမှုလိုအပ်တဲ့ စက်ပစ္စည်းတွေ၊ တိကျမှုမြင့်သော ကိရိယာတွေကို ဒီဇိုင်းနဲ့ ထုတ်လုပ်သည့် ပညာရပ်ဖြစ်ပါတယ်။ CNC Machining, Robotics, Optics, Semiconductor Manufacturing စတဲ့ နယ်ပယ်တွေမှာ အသုံးပြုပါတယ်။ လေယာဉ်, မိုဘိုင်းဖုန်း, ကွန်ပျူတာချစ်ပ်နဲ့ ဆေးဘက်ဆိုင်ရာ ကိရိယာတွေထုတ်လုပ်ရာမှာ အရေးကြီးပါသည်။

Career Path

1. Precision Machine Designer
2. CNC Machining Engineer
3. Robotics Engineer
4. Semiconductor Process Engineer
5. Optical Instrumentation Engineer
6. Quality Control Specialist
