Available Majors
Software Engineering ▶️

Software Engineering ဆိုတာက Computer ကိုအသုံးပြုပြီး လူသုံးနိုင်တဲ့ Application, Web, Mobile Software တွေကို စနစ်တကျရေးသားတည်ဆောက်တဲ့နယ်ပယ်ပါ။ တကယ်တော့ Programming တစ်ခုပဲမဟုတ်ဘဲ Large-Scale Software Projects တွေကို plan, build, test, maintain လုပ်နိုင်ဖို့ အဓိကထားသင်ကြားရတဲ့ ဘာသာရပ်ဖြစ်ပါတယ်။

Career Path

1.Software Developer / Engineer
2.Full Stack Developer (Frontend + Backend)
3.Mobile App Developer (Android/iOS)
4.Web Developer
5.QA Engineer / Software Tester
6.DevOps Engineer
7.UI/UX Designer (optional)

Knowledge Engineering / Artificial Intelligence▶️

လူတွေထဲက expert knowledge ကို စုဆောင်းပြီး၊ computer system (AI, Expert Systems, Knowledge-based Systems) မှာ အသုံးချနိုင်အောင် စီမံထိန်းချုပ်ပေးတဲ့ computer science ဘာသာရပ်ဖြစ်ပါတယ်။AI သည် လူ့ဉာဏ်အတိုင်းစဉ်းစားနိုင်စေဖို့ Data တွေသုံးပြီး လေ့လာ၊ ခန့်မှန်း၊ ဆုံးဖြတ်မှုလုပ်နိုင်တဲ့နည်းပညာပါ။ ChatGPT လို model တွေကလည်း AI နဲ့ပတ်သက်ပါတယ်။

Career Path

1.AI Engineer
2.Machine Learning Developer
3.Data Scientist / Data Analyst
4.NLP Specialist (e.g. chatbot, translation)
5.Research Engineer (AI field)

Business Information Systems ▶️

BIS သည် စီးပွားရေးလုပ်ငန်းများကို IT နည်းပညာဖြင့် ပိုမိုထိရောက်စေဖို့ရည်ရွယ်ပါတယ်။ စီမံခန့်ခွဲမှု၊ CRM Systems, ERP Systems စတဲ့ စီးပွားရေးစနစ်များကို အဓိကထား လေ့လာကြပါတယ်။

Career Path

1.Business Analyst
2.IT Consultant
3.ERP System Developer
4.CRM Specialist
5.Project Manager
6.Digital Transformation Officer

High Performance Computing▶️

HPC သည် ဘယ်လိုကြီးမားတဲ့ Data မဆို မြန်မြန်ဆန်ဆန်တွက်ချက်နိုင်ဖို့အတွက် Supercomputer, Cloud, Parallel Systems တို့ကိုအသုံးပြုပါတယ်။ Climate simulation, Genetics analysis, AI training တွေမှာအသုံးများတယ်။

Career Path

1.HPC Software Engineer
2.Scientific Programmer
3.Simulation Developer
4.Big Data Engineer
5.Cloud Performance Tuner

Embedded Systems▶️

ဖုန်း၊ကား၊အိမ်သုံးစက်ပစ္စည်းတွေထဲမှာ တပ်ဆင်ထားတဲ့ စနစ်တွေ၊ ရေးသားတည်ဆောက်မှုတွေကို အဓိကထား လေ့လာကြပါတယ်။

Career Path

1.Embedded Systems Engineer
2.Firmware Developer
3.IoT Developer
4.Robotics Programmer
5.Automotive Embedded Software Engineer

Computer Communication & Networks▶️

Computer များအချင်းချင်း ဆက်သွယ်နိုင်ဖို့၊ Internet အသုံးပြုနိုင်ဖို့ အရေးကြီးတဲ့နည်းပညာဖြစ်ပါတယ်။ Network setup, Security, Protocols တွေကို အဓိကထား လေ့လာကြပါတယ်။

Career Path

1.Network Administrator
2.Network Engineer
3.Security Analyst / Firewall Expert
4.Cloud Network Technician
5.Wireless Communication Engineer

Cyber Security▶️

Cyber Security ဆိုတာကတော့ ကွန်ပျူတာစနစ်များ၊ ကွန်ယက်များ (Networks)၊ ဒေတာများ (Data)၊ ဆာဗာများ (Servers) တို့ကို hackers, viruses, attacks စတဲ့ အန္တရာယ်တွေဆီကကာကွယ်ဖို့ လုပ်ဆောင်တဲ့ နည်းပညာနယ်ပယ်ပါ။

Career Path

1.Cybersecurity Analyst
2.Penetration Tester (Ethical Hacker)
3.SOC Analyst
4.Security Engineer
5.Incident Responder
6.Information Security Officer
7.Vulnerability Analyst
8.Application Security Analyst
9.Cloud Security Specialist
10.Network Security Engineer
11.Security Architect
12.Digital Forensics Analyst
13.Threat Intelligence Analyst
14.Malware Analyst
15.DevSecOps Engineer, etc.
