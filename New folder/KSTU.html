Technological University (Kyaukse)

📍 On the Kyaukse–Myittha Road, near Kyaung Phyu Village, Kyaukse Township, Mandalay Region, Myanmar 📍

Available Majors
Civil Engineering ▶️

အင်ဂျင်နီယာလို့ပြောလိုက်ရင်အများစုကCivilကိုပဲအဓိကပြေးမြင်တတ်ကြပါတယ်။အဆောက်အအုံတွေ ဆောက်တဲ့အင်ဂျင်နီယာပေါ့။တကယ်တော့Civil Engineering major ဆိုတာ အိမ်ဆောက်လို့ ရတဲ့ နည်းပညာတခုထဲတော့ မဟုတ်ဘူးပေါ့နော်။ မြန်မာလိုဘာသာပြန်ရင်မြို့ပြလို့အဓိပ္ပါယ်ရတဲ့အတွက်လူသားတွေတည်ဆောက်နိုင်တဲ့ရုပ်ဝတ္ထုယဉ်ကျေးမှုတွေနဲ့အလုံးစုံသက်ဆိုင်ပါတယ်။အဆောက်အအုံတွေဆောက်တာအပြင်၊လမ်းတံတားတွေ၊ဆစ်မြောင်းတာတမံတွေ၊‌ရေပေးဝေစနစ်တွေ၊ဥယျာဉ်ပန်းခြံတည်ဆောက်မှုတွေဟာမြို့ပြအင်ဂျင်နီယာဆိုင်ရာနယ်ပယ်တွေဖြစ်ပါတယ်။ အချုပ်အားဖြင့် ဆိုရင် မြို့ပြအင်ဂျင်နီယာ ဘာသာရပ်ကို လေ့လာရာမှာ 1.Structural Engineering 2.Geotechnical Engineering 3.Water Resources Engineering 4.Transportation Engineering 5.Environmental Engineering 6.Construction Engineering ဟူ၍ အဓိက ဘာသာရပ် နယ်ပယ် (၆)ခုရှိပါတယ်။

Career Path

1.Structural engineer
2.Geotechnical Engineer
3.Environmental Engineer
4.Marine Engineer
5.Engineering Manager
6.Design Engineer
7.Water Resource Engineer

Electronic Engineering (EC) ▶️

Electronic Engineering ဆိုတာ အီလက်ထရွန်းနစ် circuit, devices နှင့် communication systems များကို အခြေခံလေ့လာပြီး designing၊ testing နှင့် implementation လုပ်ငန်းများကို ပြုလုပ်သည့် ဘာသာရပ်ဖြစ်ပါတယ်။ Analog နှင့် digital systems, sensors, embedded systems များလည်း အဓိကပါဝင်သည်။

Career Path

1.Electronics Engineer
2.Telecommunications Engineer
3.Embedded Systems Developer
4.Control/System Design Engineer

Electrical Power (EP) ▶️

Electrical Power Engineering သည် power generation, transmission, distribution နှင့် power system analysis အစရှိသည့် လျှပ်စစ်စွမ်းအားနှင့် ဆိုင်သော နယ်ပယ်များကို လေ့လာသည့်ဘာသာရပ်ဖြစ်သည်။ Transformers, motors, protection systems များကိုလည်း သင်ကြားတတ်ရသည်။

Career Path

1.Power Systems Engineer
2.Substation/Distribution Engineer
3.Electrical Maintenance Engineer
4.Energy/Utility Consultant

Mechanical Engineering ▶️

Mechanical engineering ဆိုတာ စက်ပစ္စည်းများ၊ thermodynamics၊ fluid mechanics နှင့် manufacturing processes များကို လေ့လာသည့်ဘာသာရပ်ဖြစ်သည်။ Design၊ production နှင့် testing အလုပ်များတွင် အဓိက ပါဝင်သည်။

Career Path

1.Mechanical Engineer
2.Automotive Engineer
3.Manufacturing/Production Engineer
4.Maintenance & Design Engineer

Information Technology (IT) ▶️

Information Technology သည် software development, networking, databases၊ system administration နှင့် IT services များကို အာရုံစိုက်၍ သင်ကြားပေးသည့်ဘာသာရပ်ဖြစ်သည်။ Practical programming နှင့် system design များလည်း အဓိကရှိသည်။

Career Path

1.Software Developer / Programmer
2.Network / System Engineer
3.Database Administrator / Data Manager
4.Cloud / DevOps Engineer

Metallurgy (Met) ▶️

Metallurgy သည် သတ္တုများ၏ ဖွဲ့စည်းပုံ၊ properties နှင့် processing နည်းပညာများကို လေ့လာသည့်ဘာသာရပ်ဖြစ်သည်။ Materials testing နှင့် production optimization များကိုလည်း အာရုံစိုက်သည်။

Career Path

1.Metallurgical Engineer
2.Materials Scientist / Engineer
3.Process & Quality Control Engineer (Metallurgy)
4.Mining / Mineral Processing Support Roles

Mechatronics (McE) ▶️

Mechatronics သည် mechanical, electronics, control systems နှင့် computing ကို ပေါင်းစပ်၍ automation နှင့် robotics applications များ ဖန်တီးရန် အဓိကထားသည့်ဘာသာရပ်ဖြစ်သည်။

Career Path

1.Robotics Engineer
2.Automation / Control Systems Engineer
3.Mechatronics Design & Maintenance Engineer
